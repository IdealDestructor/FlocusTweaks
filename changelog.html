Changelog: Flocus Tweaks

Version 2024-04-04:

  1. Toast Notifications:
    • Integrated toast notifications for better user feedback.
    • Notifications display when Flocus Tweaks are successfully loaded.
  2. Visual Enhancements:
    • Improved visual elements by replacing the logo with a better one.
    • Fixed visual inconsistencies by removing unnecessary classes from the off-canvas content.
    • Removed advertisements for Plus subscription and upgraded buttons.
  3. Priorities Tab:
    • Added functionality to update emoji based on priority changes (Note: this feature is not fully functional yet).
  4. Music Player Settings:
    • Replaced the existing input box with a new one for better usability.
    • Removed the disabled attribute from the save button, allowing users to update custom playlists seamlessly.
    • Implemented the feature to update the Spotify playlist based on the provided URL.
  5. Settings Tab:
    • Replaced the Support Tab with a dedicated Flocus Tweaks Settings tab for better organization.
    • Added the ability to paste background image links and save them (Note: this feature is not fully implemented yet).
  6. General:
    • Ensured compatibility with Flocus website (https://app.flocus.com/).
    • Improved code readability and structure for easier maintenance.
